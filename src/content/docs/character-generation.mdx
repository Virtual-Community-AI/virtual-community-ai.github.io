## Run

<ol class="list-decimal ml-6">
  <li>
  Prepare the OPENAI GPT-4o API key and create `.api_keys.json` under the root directory. Then, copy the following into `.api_keys.json`:
  ```
  {
    "all": [
      {
          "AZURE_ENDPOINT": "YOUR ENDPOINT HERE",
          "OPENAI_API_KEY": "YOUR KEY HERE"
      }
    ]
  }
  ```
  </li>
  <li>Open `scripts/generate_characters_all.sh` and modify the scene(s) you want to generate characters, for example, if you want to generate characters for TORONTO:
`SCENES=(TORONTO)`</li>
  <li>Run `bash scripts/generate_characters_all.sh` to start generation process</li>
</ol>

### Reminders

The script supports annotating multiple scenes at once, for example, `SCENES=(TORONTO ZURICH)`, but we recommend annotating one scene at a time.
